Gentelella Admin Bootstrap Template on Rails 5

December 7, 2016 - 1 minute read -
rails ruby admin

Rails 5 implementation of the awesome Gentelella Bootstrap admin template.

  • ruby 2.3.0
  • Gentelella 1.3.0

Setup:

  1. Clone this repo : git clone https://github.com/iogbole/gentelella_on_rails.git

  2. cd gentelella_on_rails

  3. npm install -g bower

  4. bower install

  5. bundle install

  6. rails s

That’s it!

Navigate to http://localhost:3000

If you’re using devise, you’d need to create another layout to handle authentication and sign_ups as discussed in issue #2.
